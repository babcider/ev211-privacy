EV211 설치 및 사용 안내

EV211송수신기는 같은 로컬 네트워크에서 음성 방송과 무전 통신을 제공하는 앱입니다. 대부분의 사용은 폰 서버 모드만으로 충분하며, 다중 사용자·장시간 운용이나 관리자 대시보드·채널 녹음이 필요할 때 로컬 서버 모드(Ubuntu)를 선택적으로 구축합니다.

EV211 로컬 서버란

EV211 로컬 서버는 사용자가 직접 관리하는 Ubuntu 서버입니다. 음성 데이터는 설정된 로컬 네트워크 안에서 전달되며, EV211의 다른 온라인 서비스와 독립적으로 작동합니다.

로컬 서버가 없어도 폰 서버 모드를 사용할 수 있습니다. 여러 사용자가 장시간 안정적으로 접속하거나 관리자 대시보드와 채널 녹음이 필요한 경우 Ubuntu 서버 사용을 권장합니다.

준비 사항

  • Ubuntu가 설치된 PC 또는 미니 PC
  • Docker 및 Docker Compose
  • 앱을 사용할 휴대폰과 같은 공유기 또는 LAN
  • 서버에 고정된 내부 IP 또는 DHCP 예약
  • 최소 2GB 메모리와 충분한 녹음 저장 공간

설치 예시

git clone https://github.com/babcider/ev211-field-server.git
cd ev211-field-server
sudo ./scripts/install.sh
sudo ev211ctl credentials
sudo ev211ctl doctor

네트워크 설정

서버 방화벽과 공유기 내부망에서 다음 포트를 허용합니다.

  • TCP 80, 8443
  • TCP 7880, 7881
  • UDP 50000–60000
외부 인터넷으로의 포트 포워딩은 기본 설치 방법으로 권장하지 않습니다. EV211은 내부망 사용을 전제로 설계되었습니다.

앱 연결 방법

  1. 휴대폰과 Ubuntu 서버를 같은 Wi-Fi에 연결합니다.
  2. 앱에서 Local server 모드를 선택합니다.
  3. 서버 주소에 https://서버-IP:8443 을 입력합니다.
  4. 관리자가 제공한 송신 비밀번호를 입력합니다.
  5. 마이크와 로컬 네트워크 권한을 허용합니다.
  6. 채널을 선택하고 송신을 켭니다.

관리자 안내

관리자는 https://서버-IP:8443/admin.html 에서 방송 채널과 무전기 채널을 확인할 수 있습니다. 채널별 녹음을 시작하면 서버에서 녹음이 계속되며, 종료된 녹음은 MP3로 내려받을 수 있습니다.

녹음을 시작하기 전에 참가자에게 녹음 사실과 보관 기간을 반드시 고지해야 합니다. 녹음 파일은 관리자 서버에 저장되며, 개발사는 이 파일에 접근하지 않습니다.

서버 없이 바로 쓰기 (폰 서버 모드)

  1. 두 대 이상의 휴대폰을 같은 Wi-Fi에 연결합니다.
  2. 호스트 기기: Phone as server → Phone-server walkie-talkie → Start on this phone → Start walkie-talkie
  3. 참가 기기: Phone as server → Phone-server walkie-talkie → Join a walkie-talkie, 또는 호스트 화면의 QR 코드를 스캔
  4. 마이크 권한을 허용하고 송신 버튼을 누르면 같은 그룹 참가자에게 음성이 전달됩니다.
