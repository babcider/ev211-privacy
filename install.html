EV211 설치 및 사용 안내

EV211송수신기는 같은 로컬 네트워크에서 음성 방송과 무전 통신을 제공하는 앱입니다. 대부분의 사용은 폰 서버 모드만으로 충분하며, 최대 15개 채널·관리자 대시보드·채널별 녹음이 필요할 때 로컬 서버 모드(Ubuntu)를 선택적으로 구축합니다.

로컬 서버의 전체 소스와 상세 문서는 공개 저장소에 있습니다. 이 페이지는 요약이며, 최신·상세 절차는 저장소 문서를 기준으로 합니다.
저장소: github.com/babcider/ev211-field-server (Apache-2.0)

EV211 로컬 서버란

사용자가 직접 관리하는 Ubuntu 서버로, 인터넷 연결 없이 로컬 네트워크에서 LiveKit 음성 중계·채널 관리 API·관리자 화면이 동작합니다. EV211 클라우드나 개발사 서비스와 독립적으로 작동하며, 앱의 폰서버접속 모드는 이 서버를 사용하지 않습니다.

권장 환경

  • Ubuntu Server 24.04 LTS 또는 22.04 LTS (64비트 x86_64)
  • 4코어 CPU, 메모리 8GB 이상, 여유 공간 20GB 이상
  • 서버는 가급적 공유기에 유선으로 연결
  • 모든 앱 단말과 서버가 같은 사설 네트워크에 위치
  • 공유기의 AP 격리(클라이언트 격리) 기능 끄기
  • DHCP 예약으로 서버 IP 고정

빠른 설치

먼저 Docker 공식 Ubuntu 안내에 따라 Docker Engine과 Docker Compose 플러그인을 설치합니다(Docker Desktop이 아니라 Ubuntu용 Docker Engine).

sudo apt update
sudo apt install -y git curl openssl
sudo git clone https://github.com/babcider/ev211-field-server.git /opt/ev211-field-server
cd /opt/ev211-field-server
sudo ./scripts/install.sh

설치가 끝나면 서버 IP·비밀번호를 확인합니다.

sudo ev211ctl doctor
sudo ev211ctl credentials

credentials에 표시되는 서버 IP·송신 비밀번호·관리자 비밀번호는 운영 담당자에게만 전달합니다. 일반 청취자에게는 서버 IP만 안내하면 됩니다. Docker 설치와 방화벽을 포함한 전체 절차는 저장소의 설치 안내(docs/INSTALL.md)를 따르세요.

운영 명령

sudo ev211ctl status
sudo ev211ctl doctor
sudo ev211ctl logs
sudo ev211ctl restart
sudo ev211ctl backup
sudo ev211ctl update

설정과 자동 생성 비밀정보는 /etc/ev211-field/ev211.env(권한 600)에 저장됩니다. 업데이트 전 ev211ctl update가 자동으로 백업합니다.

앱에서 접속 (로컬 서버 모드)

  1. EV211을 열고 로컬서버접속을 선택합니다.
  2. 설치 시 표시된 서버 IP를 입력합니다. https://나 포트는 붙이지 않습니다.
  3. 청취자는 수신 접속을 누르고 원하는 채널을 선택합니다.
  4. 송신자는 송신자이신가요?를 누르고 송신 비밀번호를 입력합니다.
  5. 무전기는 무전기(인터컴) 모드를 누르고 송신 비밀번호를 입력합니다.
  6. 관리자는 화면 아래 관리자에서 관리자 비밀번호로 접속합니다.

처음 연결할 때 앱은 서버의 자체 서명 인증서를 기억합니다. 서버를 재설치해 인증서가 바뀌면 경고가 나타나며, 현장 관리자가 재설치 사실을 확인한 뒤에만 다시 신뢰해야 합니다.

관리자 녹음 안내

관리자 웹에서 방송·무전기 채널을 모니터링하고 채널별 MP3 녹음을 서버에 저장할 수 있습니다. 녹음 파일은 관리자 서버에 저장되며 개발사는 접근하지 않습니다. 녹음을 시작하기 전에 참가자에게 녹음 사실과 보관 기간을 반드시 고지하세요. 자세한 사용법은 저장소의 녹음 안내(docs/RECORDING.md)를 참고하세요.

서버 없이 바로 쓰기 (폰 서버 모드)

Ubuntu 서버 없이 같은 Wi-Fi의 휴대폰끼리 바로 무전을 사용할 수 있습니다.

  1. 두 대 이상의 휴대폰을 같은 Wi-Fi에 연결합니다.
  2. 호스트: 폰서버접속 → 폰서버 무전기 → 이 폰에서 시작
  3. 참가: 폰서버접속 → 폰서버 무전기 → 무전기 참가, 또는 호스트 화면의 QR 코드 스캔
  4. 마이크 권한을 허용하고 송신 버튼을 누르면 같은 그룹 참가자에게 음성이 전달됩니다.
